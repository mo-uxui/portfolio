COMP213 - Web Interface Design

This is the list of Elements with examples:

  • Heading Element -
    Example: <h6>This is a Heading</h6>
    Output:
    This is a Heading
  • Paragraph Element -
    Example: <p>This is a paragraph of text.</p>
    Output:

    This is a paragraph of text.

  • Line Break and Horizontal Rule -
    Example: Line one <br> Line two <hr>
    Output:
    Line one "line break"
    Line two "Horizontal Line"
  • Blockquote Element -
    Example: <blockquote>This is a quoted text.</blockquote>
    Output:
    This is a quoted text.
  • Phrase Element -
    Example: <em>Italic text</em>
    <strong>Bold text</strong>
    <code>Inline code</code>
    Output:
    Italic text
    Bold text
    Inline code
  • Unordered List -
    Example: <ul> <li>Laptop</li> <li>Phone</li> </ul>
    Output:
    • Laptop
    • Phone
  • Ordered List -
    Example: <ol> <li>First</li> <li>Second</li> </ol>
    Output:
    1. First
    2. Second
  • Description List -
    Example: <dl> <dt>HTML:</dt> <dd>Markup language for web pages</dd> </dl>
    Output:
    HTML:
    Markup language for Web pages Design
  • Special Entity Characters -
    Example: <p><div> means "division"</p> <p>& copy; 2025 MySite</p> "No space between '&'and'copy;'"
    Output:

    <div> means "division"

    © 2025 MySite

  • HTML Syntax Validation -
    Example: <p>Hello World</p>
    Output:

    Hello World

  • Structural Elements -
    Example: <header>Header Section</header> <nav>Navigation Links</nav> <main>Main Content</main> <footer>Footer Info</footer>
    Output:
    Header Section
    Navigation Links
    Main Content
    Footer Info
